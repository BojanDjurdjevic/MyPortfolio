To Do List

Simply To Do List


This is the App made only on frontend, there is no database, but all tasks are saved in locale storage.
The user can choose if the task is scheduled or not, and add it on the list.
There are 3 lists: unscheduled, scheduled and completed tasks.
If the task is scheduled, the user must enter the deadline and the App will check every second if the deadline is passed.
Then the scheduled task becomes completed automatically.
The user can mark the task as completed or delete it from the list.
